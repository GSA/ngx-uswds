Ngx USWDS Formly

Follow these steps to get started with USWDS Formly.

1.Install Formly packages:

npm install @ngx-formly/core
npm install @gsa-sam/uswds-formly

Once installed, `FormlyModule` and `UsaFormlyModule` will be imported in the Module:

There are different formly types available for the uswds-formly, please click on the side navigation for each formly types more.
  • Checkbox
  • Input
  • Search
  • Radio button

For more implementation and configuration visit Ngx-formly
