General Accessibility

The modal is dynamically opened outside the normal tab flow of the page and traps focus within the modal when opened.

The other elements within the page body have aria-hidden set to true when a modal is opened

Users may pass in aria-labelledby or aria-describedby attribute values through model config when opening the modal

The modal will return focus to opener element if the opener was in focus for all browsers. However, that behaviour is inconsistent across browsers if the modal was opened through a click and the opener button was not in focus on click

Through keyboard, pressing the Esc button will also close the modal (This behavior can be toggled off if needed)
