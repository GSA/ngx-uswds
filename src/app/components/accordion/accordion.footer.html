General Accessibility

Each accordion header uses an element with role="heading", with default heading level of 4. Users may pass in the heading level through [headerLevel] property in usa-accordion. This level will be applied to all accordion headers in that accordion group.

Each header also uses a button with [aria-expanded] property to indicate whether the panel is expanded or collapsed.

If the header is multi-selectable, then [aria-multiselectable] attribute will be applied.

The accordion internally uses an incrementing id to ensure uniqueness between each accordion header's ids. Users may pass in their own id

The accordion uses display: none style to hide collapsed accordions

Keyboard Interaction

Key Function
Tab / Shift + Tab Move focus to previous and next focusable item
Enter/Space Open / close accordion
Up arrow / Down arrow Move focus to previous and next accordion item in group
Home / End Move focus to first / last focusable accordion item in group
