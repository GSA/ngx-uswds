The radio group component allows a name value to be passed in - This name value is applied to all radio buttons within the group for Accessibility. It allows browsers and screen readers to understand which set of radio buttons belongs in which group

Each radio button will have it's aria-checked set to true when checked and false when it becomes unchecked

Only one radio button within a radio group can be focused in through tab key press. Once focused, users may select options within the radio group using the arrow keys

Screen readers should announce total number of radio buttons within the group as well as the position of the currently focused radio button as user moves focus throughout the different buttons

Keyboard Support

Key Function
Tab
  • Moves keyboard focus to the checked radio button in a radiogroup.
  • If no radio button is checked, focus moves to the first radio button in the group.
Space
  • If the radio button with focus is unchecked, it's state will be changed to checked.
Down arrow
  • Moves focus to next radio button in the group.
  • If focus is on the last radio button in the group, move focus to the first radio button.
Up arrow
  • Moves focus to previous radio button in the group.
  • If focus is on the first radio button in the group, move focus to the last radio button.
