For each UsaStepIndicatorModel, you can define the properties segmentScale and completionPercent. segmentScale defines the width of the segment relative to other segments. Scale values can be 0.5, 1, 1.5, 2, 2.5, 3, 3.5, or 4. completionPercent defines partial completion of current step. By default, each step is either 0% complete or 100% complete. However, if completionPercent is defined, the width of the current step segment will be adjusted to visually indicate such completion.

*Development Note: The Back and Next buttons are NOT part of the step indicator and should be implemented separately
