General Accessibility

The submenu buttons contain aria-expanded attributes that toggles between true and false depending on the submenu is open or closed

The overall header is wrapped inside a header tag, and the navigation items are wrapped inside nav element

The default links when a template is not provided are build with a tags, allowing users to tab through the header

When the menu opens in mobile mode, focus automatically moves to the opened menu's close button, and then moves back to the menu button on close
