General Accessibility

Tables can have up to two rows of header with scope="col" or scope="row" Column row headers can be defined with usaHeaderRowDef attribute. Row headers can be attached as rowHeader property to each TableDataSource

Users are allowed to define a caption inside the table tag to indicate things like title, attribution, or a last-updated date

An aria-live region is created for each table to allow screen readers to announce whenever sorting data is modified

Aria label attributes are automatically added to column headers and sortable buttons when sorting is enabled
