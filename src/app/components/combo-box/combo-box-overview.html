General Accessibility

The combobox input contains role="combobox", with aria-expanded attribute that defines whether the list box is open or not. This is paired with aria-owns property to allow accessibility tools awareness of textbox to listbox matches

The combobox also has aria-autocomplete="list" to indicate that the autocomplete behavior of the text input is to both show an inline completion string and suggest a list of possible values in a popup

The combobox internally uses an incrementing id for both the textbox and the listbox in order to ensure there is no overlap in id between multiple combobox in the same page. Users map pass in their own id

Keyboard Interaction

Key Function
Down Arrow
  • If focus is on textbox, focus first item in listbox
  • If focus is on list box, focus next item in listbox
Up Arrow
  • If focus is on textbox, close the listbox
  • If focus is on list box:
    • If focus is on first item, focus on textbox and close listbox
    • If focus is not on first item, focus on previous item in listbox
Tab
  • Selects focused item if focus is on listbox
  • Moves to next focusable element if focus is on textbox
Enter Selects focused item if focus is on listbox
Home / End Move focus to first / last item if focus is on listbox
