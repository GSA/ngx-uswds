The top level element of step indicator uses aria-label="progress" to help provide context to screen readers

The step indicator uses a role="tablist" for the wrapper element of all step segments and role="tab" for each segment.

The indicator interlly uses an incrementing id to ensure uniqueness between each indicator and segments' id. Users may pass in their own id.

The indicator contains additional text for screen readers to indicate completed and current step as users select through each segment.
